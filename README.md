## setup instructions

Before run this site you have to 

-instalar php7
-instalar MYSQL8
-install composer
-instalacion nodejs
-instalacion Git

Once you have cloned the project run 

    cp .env example .env
    php  artisan key:generate

add db info to .env

    DB_DATABASE=<replaceme>
    DB_USERNAME=<replaceme>
    DB_PASSWORD=<replaceme>

them run

    composer install
    npm install
    npm run dev
## How to run the app
    
    php artisan serve --host=0.0.0.0
